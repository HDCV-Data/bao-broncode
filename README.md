# bao-broncode
Buitenlandse Zaken verstrekt Schengen KVV en gebruikt daarvoor een datagedreven werkwijze, het Informatie Ondersteund Beslissen (IOB). Een onderdeel van IOB is de Buitenlandse zaken Analyse Omgeving (BAO) waarbij aanvragen worden vergeleken met bronnen en profielen. De beslisboom die deze profielen maakt wordt hier openbaar weergegeven.
